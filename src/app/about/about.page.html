About

This app helps to find out if there are dangerous potholes or major potholes nearby.

Accidents

Accidents are found by looking for yellow markers with an 'A' on them. You can add a new accident to the map by clicking on the accident marker in the top right hand corner. Once clicked, a marker will appear on your location which then can be dragged to the desired location. This markers are only visible for one hour. You can view all accident markers from the past two weeks by clicking on the check box in the top left hand corner of the map.

Potholes

Potholes work the same as accident. The marker can also be found in the top left hand corner. It is a orange marker with the letter 'P' on it. Pothole markers can have information windows on them, which may display information about the pothole. Only admin can alter this information.

There can also be fixed pothole markers on the map. These are green markers with the letter 'P' on them. This means that the pothole is fixed and not dangerous any more.

Search Bar

If you would like to see potholes or accidents that may be in a different location than where you currently are, you can search using the search bar at the top of the screen. Please enter in the address you would like to search, such as "Galway, Ireland" and then click "GO". This will bring you over to Galway.
